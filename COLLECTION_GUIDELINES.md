# 꿀템장바구니 데이터 수집 지침서

## 1. 수집 목표

**앱의 목적**: 유튜버들이 추천하는 오프라인 매장(다이소, 이케아, 코스트코) 상품을 정리하여 사용자가 쇼핑할 때 참고할 수 있게 함

**핵심 원칙**: 영상에서 **실제로 긍정적으로 추천한 상품**만 수집

---

## 2. 영상 선별 기준

### 2.1 수집해야 하는 영상 ✅

| 기준 | 설명 | 예시 |
|------|------|------|
| 추천/리뷰 영상 | 특정 매장 상품을 추천하는 영상 | "다이소 꿀템 TOP 10" |
| 하울 영상 | 구매한 상품을 소개하는 영상 | "이케아 하울! 30만원어치 샀어요" |
| 비교 리뷰 | 상품 비교 후 추천하는 영상 | "다이소 vs 올리브영 화장품 비교" |
| 시즌 추천 | 계절/이벤트별 추천 | "여름 필수템! 코스트코 추천" |

### 2.2 제외해야 하는 영상 ❌

| 기준 | 설명 | 예시 |
|------|------|------|
| 비추천 영상 | 사지 말라는 영상 | "다이소에서 절대 사면 안되는 것" |
| 매장 방문 브이로그 | 상품 추천 없이 매장만 둘러보는 영상 | "다이소 구경 브이로그" |
| 뉴스/이슈 | 매장 관련 뉴스 | "다이소 화재 사고" |
| 다른 매장 내용 | 검색은 다이소인데 내용은 올리브영 | 제목에 다이소 있지만 올리브영 제품 리뷰 |

### 2.3 영상 품질 기준

```
최소 조건:
- 조회수: 1,000회 이상
- 자막 길이: 300자 이상
- 상품 언급: 최소 3개 이상
- 가격 언급: 최소 1개 이상
```

## 3. 상품 추출 기준

### 3.1 수집해야 하는 상품 ✅

| 조건 | 설명 |
|------|------|
| 긍정적 추천 | "이거 진짜 좋아요", "강추", "꼭 사세요" |
| 구체적 상품명 | "스텐 배수구망", "KALLAX 선반" |
| 가격 언급 | "2천원", "3,000원", "만원대" |
| 해당 매장 상품 | 다이소 영상이면 다이소 상품만 |

### 3.2 제외해야 하는 상품 ❌

| 조건 | 설명 | 예시 |
|------|------|------|
| 부정적 언급 | 비추천, 별로, 사지 마세요 | "이 선풍기는 별로예요" |
| 다른 매장 상품 | 다이소 영상에서 올리브영 제품 언급 | "올리브영 립틴트도 좋아요" |
| 불명확한 상품 | 구체적 상품명 없음 | "그냥 뭐 이것저것" |
| 비교 대상 | 추천 상품과 비교용으로만 언급 | "A보다 B가 나아요"의 A |

### 3.3 상품 데이터 필수 필드

```json
{
  "name": "스텐 배수구망",        // 필수: 구체적 상품명
  "price": 2000,                // 필수: 원 단위 숫자
  "category": "주방",           // 필수: 카테고리
  "reason": "물때 안 끼고 위생적", // 필수: 추천 이유
  "confidence": 0.95,           // 필수: AI 신뢰도 (0.7 이상만 저장)
  "is_recommended": true        // 필수: 긍정적 추천 여부
}
```

## 4. 매장별 특수 규칙

### 4.1 다이소 (daiso)

**검색 키워드**: "다이소 추천", "다이소 꿀템", "다이소 하울"

**가격 범위**: 1,000원 ~ 10,000원 (대부분 5,000원 이하)

**주의사항**:
- 올리브영/세리아/미니소 상품 제외
- "다이소에서 산 000" = 다이소 상품
- "다이소처럼 저렴한" = 다이소 상품 아님

**카테고리**:
- 주방용품, 욕실용품, 수납/정리, 문구, 뷰티, 시즌상품

### 4.2 이케아 (ikea)

**검색 키워드**: "이케아 추천", "이케아 꿀템", "이케아 가구"

**가격 범위**: 1,000원 ~ 500,000원

**주의사항**:
- 상품명이 스웨덴어 (예: KALLAX, MALM, LACK)
- 조립 난이도 언급 시 함께 기록
- 한글 상품명과 스웨덴어 상품명 둘 다 저장

**카테고리**:
- 가구, 수납, 주방, 조명, 텍스타일, 장식

### 4.3 코스트코 (costco)

**검색 키워드**: "코스트코 추천", "코스트코 꿀템", "코스트코 하울"

**가격 범위**: 5,000원 ~ 200,000원

**주의사항**:
- 멤버십 전용 상품 (일반인 구매 불가 표시)
- 수입 상품 브랜드명 주의 (커클랜드 등)
- 식품 유통기한 관련 내용 제외

**카테고리**:
- 식품, 생활용품, 가전, 의류, 주방

## 5. 데이터 검증 체크리스트

### 5.1 영상 레벨 검증

- [ ] 영상 제목에 해당 매장명 포함?
- [ ] 자막 내용이 300자 이상?
- [ ] 상품 추천 내용이 있는가?
- [ ] 비추천/비판 영상이 아닌가?

### 5.2 상품 레벨 검증

- [ ] 상품명이 구체적인가? (XX 브랜드 OO 제품)
- [ ] 가격이 해당 매장 범위 내인가?
- [ ] 긍정적 추천인가? (비추천 제외)
- [ ] 해당 매장 상품이 맞는가? (다른 매장 상품 제외)

### 5.3 카탈로그 매칭 검증

- [ ] 카탈로그에 유사 상품 존재?
- [ ] 가격 차이 ±30% 이내?
- [ ] 카테고리 일치?

## 6. AI 프롬프트 가이드라인

### 6.1 상품 추출 프롬프트 핵심

```
1. 영상에서 **긍정적으로 추천**하는 상품만 추출
2. 비추천/비교대상 상품 제외
3. 해당 매장({store_name}) 상품만 추출
4. 구체적 상품명과 가격 필수
5. 추천 이유를 영상 내용에서 인용
6. 신뢰도 점수 0.0~1.0 부여
```

### 6.2 필터링 키워드

**긍정 키워드** (추출 대상):
- 추천, 강추, 꿀템, 필수템, 인생템
- 좋아요, 대박, 최고, 가성비
- 꼭 사세요, 후회 없어요

**부정 키워드** (제외 대상):
- 비추, 별로, 후회, 실패
- 사지 마세요, 돈 아까워요
- 안 좋아요, 품질 별로
